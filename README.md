JSON.NET 抽象型プロパティ シリアライズサンプル
===============================

これはなに？
-------------------------------

> JSON.NET でインターフェイスなどの抽象型のプロパティを持つオブジェクトをシリアライズ／デシリアライズしたい
> しかしながら、抽象型はインスタンスを直接作成することが出来ないため、シリアライズ出来てもデシリアライズができない
> 本テストコードでは、次のようなアプローチでその解決を試行する。


* JsonConverterを用意
* 事前にコレクションにデシリアライズ先として想定される型 (Type) を貯めておく
* 対象となるインターフェイス内で具体型の型名、あるいは識別子になる値を持たせておく
* コンバーター内のReadJsonメソッドで、JSONデータから具体型の識別子となる値を取得し、具体的な型を指定した上でデシリアライズ


既知の問題点
-------------------------------

* コンバーターのReadJsonメソッド内で具体型を指定した後にデシリアライズを行おうとした際も、同じReadJsonメソッドが実行される
* 上記のメソッドの循環呼び出しによって StackOverflow が発生してしまう。


検討中の解決策
-------------------------------

* 具体的な型が指定されている場合は、別のコンバーターが使用されるように定義する
* or JSON.net に準ずる独自のシリアライザを定義する (無理)
